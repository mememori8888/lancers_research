### まとめ

アメリカ企業リサーチはAIではなく精度が高い人的リサーチにて進行し、データ欠損時の対応方針や対象企業の拡大計画が議論された。

**リサーチ作業手順と企業選定**  
リサーチ作業手順として、入力シートから企業IDと企業名をコピーし、ルーフシートに追加するよう説明された。指標が1つも示されていない企業はリサーチ対象としないことが決定された。

**人的リサーチ選択の根拠**  
AIによるPOCが進められているものの、海外企業の報告書はフォーマットが統一されていないため、スクレイピングよりも人手による収集の方が精度が高く間違いが少ないため、人的リサーチが選択された。このデータは、自社と他社の人的データのギャップ分析やダイバーシティ推進企業と生産性の相関関係分析に活用される。

**対象企業拡大とツール検討**  
今後、リサーチ対象企業はアジアや中国などに拡大する計画があり、3月中の完了が希望された。まんから紹介された自然言語検索ツール「Bright Data」について、居原健は今回の依頼を通じてそのクオリティを確認し、導入を検討する可能性を示した。
